Note Transcriber Demo

This demo transcribes notes and displays them using VexFlow.

Note:

  1. All notes are treated as quarter notes regardless of length.
  2. Pauses between notes are not recorded.
  3. Half notes are only represented as sharps.
  4. Keys hit within 25 ms of each other are displayed as a chord.
  5. Only the last 8 keys/chords are displayed.
